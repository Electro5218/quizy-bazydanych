# 📡 API Documentation

## Authentication

### POST /auth/register

Register a new user

### POST /auth/login

Login user

### POST /auth/reset-password

Send reset token

---

## Users

### PATCH /users

Update user data

### DELETE /users

Soft delete account

---

## Groups

### POST /groups

Create group

### POST /groups/join

Join group

---

## Quizzes

### POST /quizzes

Create quiz

### GET /quizzes

Get quizzes

---

## Attempts

### POST /attempts

Submit quiz answers

### GET /attempts/:id

Get attempt results
